This is Model 1 trained at last of course 4 of Deep Learning


The model is of a trained neural network to recognize whether the person in the input picture is smiling or not. ( happy/not happy)


The .h5 model is available here


To get prediction make a POST request to this URL with key as 'image' and the image in jpg format


Sample cURL request as:

curl -X POST \ https://dlapis.herokuapp.com/smilenotsmile \
-H 'content-type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F image=@/Users/siddharth/Desktop/code/deeplearning/course1/images/cat1.jpg



You also use the webform below.


If the model runs successfully, json response with predition 0 ( unhappy ) and 1 ( happy ) and code 200.
Errors are returned with code 400
