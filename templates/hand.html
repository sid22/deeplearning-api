This is Model 2 trained at last of course 2 of Deep Learning


The model is of a trained neural network to recognize digits from a hand, the dataset was a subset of SIGNIT dataset and only recognizes hand patterns for 0,1,2,3,4 and 5. The NN is relatively primitive and hence as such works only with images without complex backgrounds.


The pickled model is available here


To get prediction make a POST request to this URL with key as 'image' and the image in jpg format


Sample cURL request as:

curl -X POST \ https://dlapi-216320.appspot.com/handdigit \
-H 'content-type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F image=@/Users/siddharth/Desktop/code/deeplearning/course1/images/cat1.jpg
